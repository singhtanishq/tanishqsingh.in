About Me

Who I Am

I'm Tanishq Singh, a software developer from Varanasi, Uttar Pradesh, India with 2+ years of hands-on experience in building modern web applications. My journey began with a deep curiosity about how technology shapes the world, which steadily transformed into a passion for crafting intuitive user experiences and robust systems.

I work extensively with modern JavaScript ecosystems—covering frameworks like React, Next.js, and Node.js—along with Selenium for automation testing, and relational/non-relational databases such as MySQL and MongoDB. I'm focused on writing clean, maintainable code while building interfaces that balance both performance and aesthetics.

My skill set also includes tools and platforms like Git, Docker, Kubernetes, GraphQL, and CI/CD workflows, helping me deliver scalable software with efficient development pipelines. Outside of work, I enjoy exploring UI/UX principles, contributing to open-source, and sharing knowledge with the developer community.
